LeagueForce.us

League Force

A web application in which the user has access to create a league, filling it with data about the teams, players, and matches. Users can also read newsletters created by the league administrator, message other users, and link social media accounts. Various libraries, frameworks, API's, were used in the course of creating this website such as JavaScript, HTML5, CSS3, Bootstrap, AngularJS, Node.js, Express, MongoDB, Mongoose, Passport.js, Google Maps API, Google Auth API, FaceBook Auth API, and Cloudinary API.


Swinder

Swinder

A web application similar to Tinder, but for swinging couples. Couples register on the website and put in their preferences. Matches are shown according to their preferences. Couples then have access to a messaging system where they can communicate with their matches. Technologies used in developement included JavaScript, HTML5, CSS3, Bootstrap, AngularJS, Node.js, Express, MongoDB, Mongoose and Passport.js.


*Currently working on hosting the app.

AIMClone

AIMClone

A text messaging application where users are able to sign up and send private messages to each other. Each conversation is kept separate. A history of all conversations between two participants is maintained between sessions. Participants are able to edit and delete conversations. Technologies used included JavaScript, HTML5, CSS3, Bootstrap, AngularJS, Node.js, Express, MongoDB, Mongoose, and Passport.js.


*Currently working on hosting the app.
